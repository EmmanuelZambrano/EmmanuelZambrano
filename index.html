Web Portal Redesign

This project was part of a Human-Computer Interaction course that was taken during the Spring of 2015. The purpose of this project was to redesign the IIT web portal. The project was a collaborative work with other students. The project was split up into different parts. First, a specific behavior had to be chosen. My team chose to redesign the way users update their personal information. Next, the current process of updating personal information was analyzed. The redesign was completed by using ideas and principles learned in class about Human-Computer Interaction.
